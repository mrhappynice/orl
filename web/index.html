OpenRadio Live Project

Built for community storytellers.

OpenRadio Live is a lightweight broadcasting stack that lets anyone run a station with minimal hardware. It combines a low-latency HLS stream, a terminal DJ tool, and simple workflows for mixing a playlist with mic and desktop audio.

The project is open source and designed for local voices: share updates, host live conversations, and keep communities connected without heavy infrastructure.

Open source & minimal setup
TUI DJ + playlist control
Mic + desktop mix support
Community-first broadcasting
